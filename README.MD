## Kurulum

Projeyi çalıştırmak için aşağıdaki adımları izleyin:

1. Bu depoyu klonlayın:
    ```sh
    git clone <repository-url>
    ```
2. Proje dizinine gidin:
    ```sh
    cd weekFourSchoolManagementSystem
    ```
3. Gerekli bağımlılıkları yükleyin:
    ```sh
    dotnet restore
    ```
